What is Weed Dog?

Weed Dog is a passion project created by myself originally using the Lua programming language
along with the Love2D framework. I have since moved onto using Unity and the pace I am able to work
on the game at is so much faster than the pace at which I was at when I had to work with Love.

Okay, but what is Weed Dog as a character?

Weed Dog is based off an image of my friend's dog who looked high, it was mere chance we got the dog
in the background, as the purpose of the image wasn't even the dog. Though it is a chance I am glad came to.
This project turned into something much bigger than it was ever thought to be and with nearly 1 year of the
of Weed Dog being thought of--at the time of this being written it is March 17, 2022. I am very lucky to have
been able to work on this project and create my vision of this game from the ground up and make an enjoyable
experience for my friends.
